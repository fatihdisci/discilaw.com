## Keystatic Admin Panel Test Makalesi

Bu içerik, sitenin yönetim panelinin doğru çalışıp çalışmadığını test etmek amacıyla oluşturulmuştur.

### Test Amacı

Bu yazı aşağıdaki süreçlerin çalışıp çalışmadığını kontrol etmek için eklenmiştir:

* Admin panelden makale oluşturma
* İçeriğin GitHub repository'ye kaydedilmesi
* Vercel üzerinde otomatik deploy gerçekleşmesi
* Yazının web sitesinde yayınlanması

### Test Sonucu

Eğer bu yazı web sitesinde görünüyorsa içerik yönetim sistemi doğru şekilde çalışıyor demektir.
